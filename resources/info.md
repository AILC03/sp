
### Cómo usarlo en Termux:

1. Abre Termux y crea el archivo:

   ```bash
   wpkg install wget
   wget http://github.com/sp/resources/info.sh
   nano info.sh
   ```
2. Pega el código y guarda con `CTRL+O`, luego `CTRL+X`.
3. Dale permisos de ejecución:

   ```bash
   chmod +x info.sh
   ```
4. Ejecútalo:

   ```bash
   ./info.sh
   ```
